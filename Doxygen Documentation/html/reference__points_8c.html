void check_reference_points ( struct odometryTrackStruct *  ot,
struct referencePos *  ref 
)

This function checks the current robot position with the previous recorded reference position. If a relation is found, the current encoder position are updated with the saved encoder postions. The parameters are a pointer to the struct odometrtTrackStruct and refrencePos and an int which defines which corner it is. The corner is defined in the run() function where this function will be called. 1 = lower_left 2 = lower_right 3 = upper_left 4 = upper_right
